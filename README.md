# C Programming

A collection of C programs covering fundamental concepts, data structures, algorithms, and problem-solving exercises. This repository is intended for learning, practice, and quick reference.

## 📂 Repository Structure

The repository contains programs organized by topic, such as:

* Basics
* Operators & Expressions
* Conditional Statements
* Loops
* Functions
* Arrays
* Strings
* Pointers
* Structures & Unions
* File Handling
* Searching Algorithms
* Sorting Algorithms
* Recursion
* Pattern Programs
* Miscellaneous Programs

> *The directory structure may evolve as more programs are added.*

## 🛠️ Requirements

* GCC Compiler (MinGW/MSYS2 or GCC on Linux)
* Visual Studio Code (optional)
* Any C compiler supporting the C99 standard or later

## ▶️ Compiling and Running

### Linux / macOS

```bash
gcc program.c -o program
./program
```

### Windows (MinGW GCC)

```bash
gcc program.c -o program.exe
program.exe
```

## 📌 Notes

* Only source code (`.c`) is maintained in this repository.
* Compiled executables (`.exe`) and editor-specific files are excluded using `.gitignore`.
* Programs are added and updated regularly as part of continuous practice.
